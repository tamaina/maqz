SVG Telopper

SVGを読み込んでOBSに良い感じのテロップを出したいと思ったので、OBSのブラウザソースを利用するテロッパーを作りました。

リポジトリ tamaina/svg-telopper on GitHub

バージョン別取扱説明書

  • SVG Telopper v2
    v2では、テロップの管理をブラウザの管理画面から行います。
  • SVG Telopper v1
    v1は、OBSのブラウザソースのURLからクエリ文字列によってテロップを設定します。
